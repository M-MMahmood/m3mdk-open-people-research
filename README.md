# M3MDK Open People Research

M3MDK Open People Research is a Windows desktop app for high-level public-web people research.

## What it does

- Searches publicly available web results by name and optional filters
- Organizes likely matches into a scored results view
- Opens sources directly in the browser
- Generates a local insight brief from current results
- Exports reports for offline review

## Key features

- Local desktop experience
- No cloud account required
- Insight Engine Lite for explainable local briefing
- Search history and quick re-run workflow
- Exportable HTML and CSV output

## Important notice

This software is provided free of charge for entertainment and informal public-web research only.

Do not use it for employment, housing, credit, insurance, licensing, or other eligibility decisions.

Results may be incomplete, outdated, or incorrect. Always verify with the original source.
